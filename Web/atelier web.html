Présentation

Le but de cet atelier, ouvert à tous, est d'apprendre les bases du langage html.
Grâce à lui, les élèves pourront créer leur propre site. Les meilleurs sites seront mis en ligne ici.

Pour créer un site, nous ne partons pas d'une page blanche. Nous allons utiliser les deux fichiers ci-dessous et allons les modifier.
La première étape est de les télécharger dans un même dossier que nous appellerons "Site" temporairement.

  • Top cinema, le fichier html
  • Top cinema, le fichier css
